Solana Limit Orders DEX

API Endpoints

The following RESTful API endpoints are available:

GET /api/health
POST /api/generate_wallet
POST /api/import_wallet
GET /api/get_balances
GET /api/get_prices
POST /api/swap_token
POST /api/set_limit_order
GET /api/list_limit_orders
POST /api/cancel_limit_order

Features

  • Create limit orders to swap tokens at specific price points
  • Set stop loss orders to protect your investments
  • Monitor token prices in real-time
  • Execute swaps using Jupiter Aggregator for best rates

Getting Started

To use this API, you will need to:

  1. Generate a wallet or import an existing one
  2. Check your token balances
  3. Create a limit or stop loss order
  4. Monitor the status of your orders
